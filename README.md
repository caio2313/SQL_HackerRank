# HackerRank SQL Solutions 🏆 

This repository contains my solutions for SQL challenges on HackerRank. I have achieved the **5-Star Gold Badge**, demonstrating proficiency in complex query logic and data manipulation.

## 🚀 Why this repository?
In Data Engineering, efficient data transformation at the source (SQL/Pushdown) is key to performance. These challenges helped me solidify the logic needed for complex ETL processes and data analysis.

## 🧠 Key Technical Challenges Solved
- **Statistical Analysis:** Calculating the **Median** in MySQL 8.0 using Window Functions (`PERCENT_RANK`).
- **Geospatial Logic:** Implementation of **Manhattan** and **Euclidean** distance formulas.
- **Advanced Aggregations:** Solving multi-level business intelligence problems.
- **Window Functions:** Mastery of `ROW_NUMBER()`, `RANK()`, and partitions.

## 📂 Project Structure
- `/basic`: Fundamental SELECTs and filtering logic.
- `/intermediate`: Challenges involving advanced Joins and Subqueries.
- `/advanced`: Complex logic, including Window Functions and Geometry.
